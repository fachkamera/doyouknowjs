# Do You Know JS?

A quiz that tests how well you know JavaScript's quirks and gotchas. Each round serves a random set of code snippets — guess the output before you run it.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- [Sanity](https://www.sanity.io) (CMS), queried via [GroqD](https://github.com/FormidableLabs/groqd)
- [Tailwind CSS](https://tailwindcss.com)
- [Shiki](https://shiki.style) for code syntax highlighting
